# Installation

## Cloning the repo
```bash
git clone https://github.com/BMI203-2033/sample_code_lecture1
cd sample_code_lecture1
```

## Running the individual scripts
```bash
python3 animals.py
python3 example_functions.py
python3 government.py
```

## Running the module
```bash
cd test_module
python3 example.py
```
